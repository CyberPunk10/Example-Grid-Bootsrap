Сначала создается контейнер (.container).
Его ширина равна ширине 12 колонок с отступами.
Если предположить, что наш сайт не имеет максимального значения по ширине, то на больших мониторах информация будет сильно растягиваться и её неудобно будет читать.
Ширина контейнера по умолчанию составляет 1140px.
Однако её можно изменить на маскимальную (как в данном примере) - во весь экран (заменив класс .container на .container-fluid.)

Внутри контейнера создаётся ряд для контента (.row).
В одном контейнере может находиться несколько рядов.
